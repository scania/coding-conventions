# Custom Elements Style Guide

Even though custom elements is a part of the W3C HTML Standard, the way we name we name and create custom elements is of importance. Standard HTML elements are namned using one word only. A Custom Element name on the other hand side, always needs to have 2 building blocks, separated by a dash. So, example of standard elements are <code><h1></code>, <xmp><input></xmp>, <div> etc etc.
